# Summary – OOP

## Take‑aways
- **Class** is the blueprint, **Instance** is the house.
- **Inheritance** lets you specialize classes.
- **Encapsulation** protects data integrity.

## Syntax recap
```python
class Child(Parent):
    def __init__(self, x):
        super().__init__()
        self.x = x
```

## Checklist
- [ ] Initialize attributes in `__init__`.
- [ ] Use `self` to access instance data.
- [ ] Use PascalCase for class names.
