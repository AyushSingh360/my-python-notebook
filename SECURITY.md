# Security Policy

## Supported Versions

Since this is an educational repository, we generally support the latest version of the content.

| Version | Supported          |
| ------- | ------------------ |
| Master  | :white_check_mark: |

## Reporting a Vulnerability

If you discover a security vulnerability within this project (e.g., in a script or tool provided), please do **not** open a public issue.

Instead, please report it by:

1.  Emailing the maintainer (if email is public on GitHub profile).
2.  Or opening a draft Pull Request with a fix/description and marking it as sensitive if possible.

We will try to address the issue as soon as possible.
