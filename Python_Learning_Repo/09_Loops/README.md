# Loops

## 1. Topic definition
Loops repeat a block of code multiple times.

## 2. Why loops exist
- Iterate over collections (lists, files).
- Repeat actions until a condition changes.
- Avoid code duplication.

## 3. Real‑world usage
- Reading lines from a file.
- Retry mechanisms for network requests.
- Rendering items in a UI list.

## 4. Key rules & syntax
| Type | Syntax |
|------|--------|
| `while` | `while condition:` |
| `for` | `for item in iterable:` |
| Breaks | `break` exits loop, `continue` skips iteration. |
| Else | `else:` runs if loop finishes without break. |
