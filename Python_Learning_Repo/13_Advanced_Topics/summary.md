# Summary – Advanced Topics

## Take‑aways
- **Generators** save memory.
- **Decorators** enhance functions cleanly.
- **Functional** style (`map`, `filter`) can be concise.
- **Exceptions** are the standard way to handle errors.

## Syntax recap
```python
[x for x in list]
lambda x: x
yield value
try: ... except: ...
```

## Checklist
- [ ] Use comprehensions over loops for simple transformations.
- [ ] Create custom exceptions for domain errors.
- [ ] Remember iterators run forward once.
