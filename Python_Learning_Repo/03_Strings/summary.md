# Summary – Strings

## Take‑aways
- **Immutable**: Cannot change items.
- **Text**: Unicode characters.
- **Methods**: `upper`, `lower`, `strip`, `split`, `join`.

## Syntax recap
```python
s = "Value"
print(s.upper())
print(f"Prop: {s}")
```

## Checklist
- [ ] Use f-strings for formatting.
- [ ] Remember you cannot assign `s[0] = 'a'`.
- [ ] Use `strip()` to clean user input.
