# Lists

## 1. Topic definition
A list is an ordered, mutable collection of items.

## 2. Why lists exist
- Store sequences of data (users, scores, filenames).
- Flexible sizing (grow/shrink dynamically).

## 3. Real‑world usage
- A shopping cart.
- Rows from a spreadsheet.
- A playlist of songs.

## 4. Key rules & syntax
| Rule | Example |
|------|---------|
| Literal | `[1, 2, 3]` |
| Indexing | `L[0]`, `L[-1]` |
| Slicing | `L[1:3]` |
| Methods | `append`, `extend`, `pop`, `sort` |
