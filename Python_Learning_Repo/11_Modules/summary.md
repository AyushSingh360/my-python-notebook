# Summary – Modules

## Take‑aways
- Any `.py` file is a module.
- Packages are folders with `__init__.py`.
- `if __name__ == "__main__":` helps write testable scripts.

## Syntax recap
```python
import mod
from mod import func
import mod as alias
```

## Checklist
- [ ] Ensure `__init__.py` exists for packages.
- [ ] Avoid circular imports.
- [ ] Use virtual environments for 3rd party modules.
